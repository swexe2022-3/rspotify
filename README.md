# RSpotify

RSpotify gemを利用したspotify apiによるwebアプリケーション.

## Specification

- Ruby 2.6.3
- Rails 5.2.8
- node.js と npm が必要です.

## Services 

-twitterのようなフォーロー機能

-フォーローの人が最近聞いている曲と自分が最近聞いている曲のランキングをプレイリストとして表示

-ランキングをspotifyのアカウントにプレイリストとして入れられる

## Deployment instructions

```bash
$ bundle install
$ rails db:migrate
$ rails s
```
